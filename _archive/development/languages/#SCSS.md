GRID
    Для того чтобы все ряды сетки (айтемы) подстраивались под самый большой айтем
    во всей сетке: 
        grid-auto-rows: 1fr;

OVERFLOW
    Если не прокручивается контент, то возможно не хватает свойства для родителя:
        .parent {
            height: 100%;
        }
    И для потомка:
        .child {
            overflow: auto;
        }
